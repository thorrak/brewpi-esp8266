22/* Overview of debug messages and ID's
23A copy of this file is included with the python script, so it can parse it to extract the log strings.
24The python script receives the messages as a few IDs and values in a JSON string.
25It uses this file to expand that to the full message.
26Not storing the full strings, but only the ID on the Arduino saves a lot of PROGMEM space.
27At startup the python script extracts the version number from this file and compares it to its own local copy.
28It will give a warning when the two strings do not match.
29*/
30
