Detailed Description

General interface for all temperature sensor devices.

Wraps up a BasicTempSensor and processes the values through several filters. The filters are either CascadedFilter or FixedFilter implementations, depending on the compile time value of the TEMP_SENSOR_CASCADED_FILTER macro.

◆ failedReadCount

int8_t TempSensor::failedReadCount
private

An indication of how stale the data is in the filters.

Each time a read fails, this value is incremented. It's used to reset the filters after a large enough disconnect delay, and on the first init.

-1 for uninitialized, >=0 afterwards.
