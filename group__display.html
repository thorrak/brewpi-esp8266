Macro Definition Documentation

◆ BACKLIGHT_AUTO_OFF_PERIOD

#define BACKLIGHT_AUTO_OFF_PERIOD   60

Turn off LCD backlight after this time. Seconds.

Set to 0 to disable and leave backlight on.

◆ BREWPI_LCD

#define BREWPI_LCD   1

Enable the LCD display and select its type.

Without this, a NullDisplay is used.

◆ DISPLAY_POLYMORPHIC

#define DISPLAY_POLYMORPHIC   0

Control inheritance type of Display.

When true, Display is made virtual and the various hardware specific drivers derive from it.

◆ DISPLAY_SUPERCLASS

#define DISPLAY_SUPERCLASS

Compile time value for the superclass for Display drivers.

If DISPLAY_POLYMORPHIC is true, the super class is defined as Display. Otherwise the superclass is empty.
