Detailed Description

Software simulated sensors & actuators for testing.

Enabled by setting the BREWPI_SIMULATE macro to 1

◆ BREWPI_SIMULATE

#define BREWPI_SIMULATE   0

Enable the simulator.

Real sensors/actuators are replaced with simulated versions. In particular, the values reported by temp sensors are based on a model of the fridge/beer.
